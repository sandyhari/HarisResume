Engineering Project

Got Graduted from Gitam Unversity in Vishakapatnam in the stream of Electronics and Communication Engineering.

  • Did a working model,of Hill descent control system for vehicles as a final year project.
    • The main functionality of the HDC system is to help the driver while traveling down a steap slopes and muddy roads. It does so by limiting the speed of the vehicle to a predetermined value. This allows the driver to concentrate only on the steering. Thus, the driver achieves better control of the vehicle. You can activate the HDC merely by pressing the button. After that, it reduces the vehicle speed to the predetermined speed without any brake input from the driver.
  • Sensors used : Vehicle speed sensor, wheel speed sensors, pedal and throttle valve position sensors.
  • Programming language used : Python
